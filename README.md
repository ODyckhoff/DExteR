## Installation
1. Clone the repository:
```bash
 git clone https://github.com/ODyckhoff/DExteR.git.git
```

2. Install dependencies:
```bash
 npm install
 ```

3. Have a cup of tea.

## Usage
With DExteR installed, simply run:
```bash
 node .
```
